# Marketing Funnel & Conversion Performance Analysis

## Objective

Analyze marketing funnel and campaign data to identify conversion patterns, customer engagement trends, and campaign performance insights using Power BI.

## Tools Used

* Power BI
* CSV Dataset
* Data Visualization
* Funnel Analysis

## Key KPIs

* Total Leads
* Converted Customers
* Non-conversions
* Conversion Rate

## Dashboard Insights

* Conversion distribution analysis
* Contact type performance
* Job-wise conversion analysis
* Campaign performance evaluation

## Features

* Interactive slicers
* Funnel performance analysis
* Marketing conversion insights
* Professional dashboard visualization
